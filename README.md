## 專案介紹

Book of dice 是提供使用者進行 TRPG 桌上角色扮演遊戲的線上平台。
在平台中，你能夠以遊戲發起人、參加者或是觀眾的身分參與遊戲進行。
平台提供基本的創立、加入、搜索及刪除房間功能，使用者可在遊戲房間中進行多人即時交流，並透過平台提供的擲骰指令進行遊戲。

專案目前為未完成狀態，僅實裝了會員系統（註冊、登入、登出）、房間管理（建立、搜索、加入、刪除）及聊天室的基本發話功能。
